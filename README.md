# Project Structure

This repository contains the code, datasets, and reports for the CaterExtract project.

## Files & Folders

- **interm_notebook.ipynb** – Main Jupyter notebook for generating email threads, extracting features, and evaluating models.

### Reports / Presentations
- **CaterExtract.pdf** – Detailed project report on feature extraction results.
- **CaterExtract.pptx** – Presentation slides summarizing the project.
- **Emails-Exrtaction-Proposal.pdf** – Original project proposal.

### Data
- **interm_dataset/** – Contains all datasets used in the project:
  - **email threads/** – Generated email threads for extraction tasks.
  - **ground truth/** – JSON files with labeled ground truth features.
  - **extracted features zero-shot/** – Features extracted using GPT-4o-mini zero-shot model.
